블록과 인라인

블록태그 block tag & 인라인 태그 inline tag
- 블록 (큰 레이아웃 개념, 폭 100%, 새로운 행 배치)
- 인라인 (작은 요소 개념, 폭 자기 자신만큼 인식, 옆으로 배치)

블록 대표 예시 태그 div
인라인 대표 예시 태그 span

div > span

div 안에 span 을 넣을수 있고

span 안에 div 는 넣을수 없다

다양한 문서 태그 종류

h1~ h6 제목 (heading)
* 제목의 중요도에 따라 6 -> 1 순으로 1에 가까워 질수록 중요도가 올라갑니다.
* 웹사이트 로고의 경우 가장 중요한 <h1> 레벨을 가집니다.

p태그 단락 (block)
*제목의 내용 용도로 주로 사용합니다.

br태그 줄바꿈 (inline)
*블록 내에서 강제 줄바꿈이 필요한 경우 사용합니다.

strong태그 중요성, 심각성, 또는 긴급성 (inline)

em태그 문장 내에서 특정 문맥의 강조할때 사용 (inline)

sub, sup태그 아래첨자, 위첨자 (inline)

s, del 교체 or 삭제 텍스트 (inline)

blockquate태그 긴 인용문 (block)

q 태그 짧은 인용문 (block)

code (inline)

address 태그 연락처, 회사소개, 고객센터 등  (block)

# 하이퍼 링크 태그

a 태그 링크태그 (block, inline)
    html <a> 요소(앵커 요소)는 href 속성을 통해 다른 페이지나 같은 페이지의 어느 위치, 파일, 이메일 주소와 그 외 다른 URL로 연결할수있는 하이퍼링크를 만듭니다.

1. 절대 경로
    * 절대 경로는 변하지 않는 주소로 쉽게 말해 흔히 알고있는 웹주소를 떠올리면 됩니다.
    naver -> 네이버가 망하지 않는 한 사라지거나 변경되지 않는 주소입니다.
    * 절대경로는 https:// 웹 프로토콜로 시작합니다.

<a href=""target=""> 링크 위치 속성
    _blank : URL로 새창

상대경로(./파일명.확장자)
    * 경로를 작성하는 파일 ((ex)html, css) 기준으로 연결하는 파일의 위치를 작성하는 것을 상대경로라 합니다.
    * ../ 상위경로?
    * ../../ 최상위 경로 올라가기 <-두번 올라가는 경우
